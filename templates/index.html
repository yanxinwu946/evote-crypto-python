同态加密

密文可直接相加，结果解密后等于明文之和。
E(m₁)·E(m₂) = E(m₁+m₂)

🤝

分布式密钥

多方参与的密钥生成和解密，需要所有参与方合作才能解密。

核心公式

加密

a = gr mod p
b = hr · gm mod p

解密

s = ask mod p
m = decode(b · s-1)

同态加法

asum = ∏ai mod p
bsum = ∏bi mod p

c = H(id, commitment)
d = a + c·sk mod q

椭圆曲线原理

曲线方程

y² = x³ + 7 (mod p)

secp256k1 是比特币使用的椭圆曲线，提供 128 位安全强度。

标量乘法

Q = k·G

椭圆曲线上的"乘法"实际上是点的重复加法，是单向函数。

安全性

已知 Q 和 G，求 k 是困难的

椭圆曲线离散对数问题（ECDLP），目前没有已知的多项式时间算法。
